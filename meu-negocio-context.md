## Medos e respostas

### "E se o site quebrar?"
- 99% dos "problemas" são simples: plugin parado, conteúdo deletado, site lento
- Solução: instalar backup automático diário (UpdraftPlus, grátis) — restaurar com 1 clique
- Usar hospedagem confiável com suporte em português (Hostgator Brasil, Kinghost)
- Cobrar manutenção mensal = fundo de seguro para ambos
- Todo problema já foi resolvido e documentado no YouTube em português
- **A verdade:** você aprende mais com o primeiro site quebrado do que com 6 meses estudando

### "Nunca fiz isso antes"
- Ninguém nasce sabendo
- O suficiente para começar é menos do que você imagina
- O cliente não quer perfeição — quer aparecer no Google
- Esperar se sentir "pronto" é a única forma real de falhar
