# 🧠 Guia Completo de IAs e Ferramentas para Estudante-Desenvolvedor

---

## PARTE 1 — IAs de Programação / Código

Estas são as mais relevantes para você. Os parâmetros de comparação são: **qualidade do código gerado, integração ao fluxo de trabalho, suporte a múltiplas linguagens, acesso ao contexto do projeto completo e custo**.

---

### 🥇 1. **Cursor** — *Editor de código com IA nativa*
**O melhor para desenvolvimento ativo do dia a dia.**

É um editor completo (baseado no VS Code) onde a IA tem acesso a todo o seu projeto, não só ao arquivo aberto. Você conversa com ele sobre o código, pede refatorações, debugging e ele entende o contexto inteiro. Para front-end, back-end e banco de dados, é hoje a ferramenta mais poderosa para quem escreve código de verdade.

**Use quando:** Estiver desenvolvendo sites, apps, APIs, qualquer coisa que envolva escrever código real.

---

### 🥈 2. **Claude (você está aqui)** — *Assistente de raciocínio e código*
**O melhor para explicações profundas, revisão de código e pesquisa técnica.**

Excelente para entender conceitos, revisar arquiteturas, depurar lógica complexa, gerar código bem estruturado e explicado. Diferente dos editores acima, não tem acesso direto ao seu projeto, mas você cola o código e conversa sobre ele. Também é o mais confiável para pesquisa baseada em fontes, pois raciocina melhor antes de responder.

**Use quando:** Quiser entender *por que* algo funciona, revisar estrutura de banco de dados, planejar arquitetura, ou pesquisar com profundidade.

---

### 🥉 3. **GitHub Copilot** — *Autocomplete de IA dentro do seu editor*
**O mais integrado ao ecossistema GitHub/VS Code.**

Funciona como um autocomplete extremamente avançado. Você começa a digitar e ele sugere o restante. Tem acesso ao seu repositório GitHub se habilitado. Boa integração com o fluxo profissional (pull requests, revisão de código). Um pouco mais limitado em diálogo do que o Cursor.

**Use quando:** Já usa VS Code e GitHub e quer sugestões em tempo real enquanto digita.

---

### 4. **Codeium** — *Alternativa gratuita ao Copilot*
Funciona de forma similar ao GitHub Copilot (autocomplete inteligente), mas tem um plano gratuito generoso. Suporta mais de 70 linguagens. Boa escolha enquanto você está aprendendo e não quer pagar.

**Use quando:** Quiser o benefício do autocomplete por IA sem custo.

---

### 5. **Phind** — *Buscador especializado em programação*
É um motor de busca voltado para desenvolvedores [Wikipedia](https://en.wikipedia.org/wiki/Manus_(AI_agent)) , que combina busca na web com geração de respostas técnicas. Muito bom para perguntas diretas do tipo "como fazer X em React" com exemplos práticos e fontes.

**Use quando:** Quiser respostas rápidas e técnicas com referências reais, como um Stack Overflow turbinado por IA.

---

### 6. **Amazon CodeWhisperer (Amazon Q Developer)** — *IA da AWS para código*
Focado no ecossistema AWS. Bom se você for hospedar em AWS, trabalhar com Lambda, S3, etc. Para iniciantes sem foco específico em AWS, tem menos vantagem que os anteriores.

**Use quando:** Seu projeto usar serviços da Amazon (hospedagem, banco de dados na nuvem AWS).

## PARTE 2 — IAs "Vibe Coding" (Geram apps inteiros por descrição)

Estas ferramentas são para quando você quer prototipar rapidamente **sem escrever muito código manual**. Úteis para aprender como um app funciona de ponta a ponta, ou para criar MVPs rápidos.

---

### 🥇 1. **Lovable** — *Melhor para apps web com UI polida*
Gera apps React completos a partir de uma descrição em linguagem natural. Integra com Supabase para banco de dados. Produz interfaces mais bonitas que concorrentes. Requer um pouco mais de conhecimento técnico para aproveitar ao máximo.

**Use quando:** Quiser prototipar um app web com visual profissional rapidamente.

---

### 🥈 2. **Base44** — *Melhor para iniciantes, tudo-em-um*
É um AI app builder que transforma qualquer ideia em um app web funcional rapidamente, para todos os níveis de habilidade. [BASE44](https://base44.com/blog/what-is-base44) Diferente do Lovable, já inclui banco de dados e gerenciamento de usuários integrados, sem precisar conectar ao Supabase separadamente. [Banani](https://www.banani.co/blog/base44-ai-review) Foi adquirido pelo Wix em meados de 2025, o que trouxe mais infraestrutura e recursos para a plataforma. [Noloco](https://noloco.io/blog/base44-platform-features)

**Use quando:** Quiser ir do zero ao app funcionando sem configurar nada. Ótimo para praticar e entender como sistemas completos se conectam.

---

### 3. **Wix ADI** — *Para sites simples e rápidos*
Focado em criar sites institucionais e de negócios com IA. Não é para apps complexos. Útil se precisar de um site de apresentação rápido.

---

### 4. **Hostinger** (com IA integrada) — *Hospedagem + criador de site*
Tem ferramentas de IA para criar sites, mas o valor principal é a hospedagem. Relevante para você na parte de colocar projetos no ar.

## PARTE 3 — IAs de Pesquisa Acadêmica

Para o seu perfil de estudante que precisa de fontes confiáveis, estas são as mais importantes.

---

### 🥇 1. **Perplexity AI** — *O melhor para pesquisa com fontes*
Responde perguntas citando as fontes usadas, com links diretos. Tem modo de busca em artigos acadêmicos. Ideal para pesquisa rápida com rastreabilidade.

**Use quando:** Precisar de uma resposta rápida e confiável com referências verificáveis.

---

### 🥈 2. **Claude** — *Melhor para síntese e raciocínio profundo*
Para pesquisas que exigem análise, comparação entre teorias, ou entender conceitos complexos. Não navega a web por padrão (a não ser que você use a versão com busca ativada), mas raciocina melhor sobre o que sabe.

---

### 🥉 3. **Elicit** — *Especialista em artigos científicos*
Ferramenta projetada especificamente para pesquisa acadêmica. Você faz uma pergunta e ela busca papers relevantes no Semantic Scholar, extraindo metodologia, resultados e conclusões. Excelente para revisões bibliográficas.

**Use quando:** Precisar encontrar artigos científicos sobre um tema específico e resumir o que cada um diz.

---

### 4. **ChatGPT** — *Versátil, mas com ressalvas*
Muito bom para explicar conceitos, gerar rascunhos e conversar sobre temas variados. Porém, mais propenso a "alucinar" (inventar informações com confiança). Para pesquisa séria, sempre valide as fontes que ele citar.

---

### 5. **Gemini (Google)** — *Bom para pesquisa integrada ao Google*
Tem acesso à busca do Google em tempo real. Bom para temas recentes. Integra bem com Google Docs, Drive e outros serviços Google que você já usa.

## PARTE 4 — IAs de Gestão de Projetos

Estas são mais corporativas e menos essenciais para um estudante individual, mas vale conhecer.

| Ferramenta | O que faz | Quando usar |
|---|---|---|
| **ClickUp** | Gerenciamento de tarefas + IA para resumir e priorizar | Para organizar seus projetos de estudo e desenvolvimento |
| **Monday** | Similar ao ClickUp, mais visual | Trabalho em equipe, menos útil solo |

Para uso solo, o **ClickUp** é mais vantajoso pelo plano gratuito generoso.

## PARTE 5 — IAs Agentes (Autônomas)

### **Manus** — *Agente autônomo de tarefas*
É um agente de IA autônomo capaz de operar de forma independente, incluindo escrever e implantar código. [Wikipedia](https://en.wikipedia.org/wiki/Manus_(AI_agent)) Como outros agentes baseados em raciocínio, consegue decompor tarefas em etapas e navegar na web de forma autônoma para obter as informações necessárias. [MIT Technology Review](https://www.technologyreview.com/2025/03/11/1113133/manus-ai-review/) Atualmente faz parte da Meta. [Manus](https://manus.im/)

Promissor, mas ainda instável para uso intenso. Ótimo para tarefas de pesquisa complexa e automação de fluxos. Guarde para experimentar quando precisar de algo que "trabalhe por você" em segundo plano.

## PARTE 6 — Ferramentas e Portais (Classificados)

### 📚 Para Pesquisa e Estudo Acadêmico
1. **Google Scholar** — Busca de artigos científicos. Gratuito, enorme acervo. Use sempre para validar informações.
2. **Elicit** — (já citado acima)
3. **DOAB** (Directory of Open Access Books) — Livros acadêmicos gratuitos e revisados. Excelente para fontes confiáveis.
4. **Internet Archive** — Arquivo histórico da web e livros. Útil para fontes antigas.
5. **Project Gutenberg** — Livros clássicos em domínio público. Gratuito.
6. **OpenStax** — Livros didáticos gratuitos e revisados por especialistas. Ótimo para exatas, física, programação básica.
7. **MIT OpenCourseWare** — Material de cursos do MIT gratuitamente. Referência de alta qualidade.
8. **Z-Library** — Acervo enorme de livros e artigos. Acesso gratuito (verificar disponibilidade na sua região).

### 🎓 Para Cursos e Aprendizado
1. **Khan Academy** — Matemática, ciências, programação. Gratuito, excelente qualidade.
2. **edX** — Cursos de universidades reais (MIT, Harvard, etc.). Muitos gratuitos para auditar.
3. **Class Central** — Buscador de MOOCs de todas as plataformas (Coursera, edX, Udemy etc.). Use para encontrar o melhor curso sobre qualquer tema.
4. **roadmap.sh** — Roteiros visuais para aprender qualquer área de desenvolvimento (front-end, back-end, DevOps, etc.). Essencial para você.
5. **Learn Anything** — Mapa de conhecimento para aprender qualquer coisa com recursos organizados.

### 🛠 Para Descobrir Ferramentas
1. **Product Hunt** — Lançamentos diários de novas ferramentas e apps. Ótimo para ficar atualizado.
2. **There's An AI For That** — Diretório de IAs organizadas por categoria/função.
3. **Alternative.to** — Encontra alternativas a qualquer software. Muito útil.
4. **StackShare** — Veja que tecnologias empresas reais usam em seus projetos.

### ⚙️ Utilidades
1. **TinyWow** — Ferramentas online gratuitas (converter PDF, editar imagem, etc.) sem instalar nada.

## 🔍 Ferramentas que Você Ainda Não Mencionou e Deveria Conhecer

- **bolt.new** — Similar ao Lovable/Base44, gera apps web completos. Muito usado pela comunidade.
- **v0 (da Vercel)** — Gera componentes React/UI a partir de descrição. Ótimo para front-end.
- **Supabase** — Banco de dados open-source como serviço. Integra com quase todos os builders de IA.
- **Netlify / Vercel** — Hospedagem gratuita para projetos front-end e full-stack. Essenciais para publicar seus projetos.
- **DeepSeek** — IA chinesa com excelente performance em código, gratuita. Vale testar.
- **Semantic Scholar** — Busca de artigos científicos com IA, complementar ao Google Scholar.
- **Zotero** — Gerenciador de referências bibliográficas gratuito. Fundamental para um estudante que faz pesquisa.

Se quiser, posso detalhar qualquer uma dessas ferramentas mais a fundo, ou montar um **plano de estudo** usando as melhores ferramentas para cada área (front-end, back-end, banco de dados, apps Android).
